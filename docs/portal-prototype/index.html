Presentación del Proyecto

El sistema web de control administrativo de ventas e inventarios Inventarysoft permite la gestión integral de los procesos administrativos y comerciales de una organización. Abarca desde la configuración y parametrización del sistema, hasta el control de productos, inventarios, ventas y clientes, asegurando la correcta administración de la información operativa. La plataforma está compuesta por módulos especializados de gestión de inventarios, ventas, clientes, seguridad y control de accesos, infraestructura del sistema, parametrización, reportes y notificaciones, los cuales operan de forma integrada para mejorar la eficiencia operativa, la trazabilidad de la información y el apoyo a la toma de decisiones.

Tablero de Historias de Usuario (HU)

Security Management
  • Como Administrador quiero gestionar roles y permisos de usuarios para controlar el acceso al sistema.
  • Como Administrador quiero registrar usuarios asociados a personas y roles para permitir el inicio de sesión.
  • Como Usuario quiero iniciar sesión en el sistema para acceder a las funcionalidades según mi rol.
Human Resources
  • Como Administrador quiero registrar empleados con sus datos personales y cargos para gestionar el staff.
  • Como Administrador quiero consultar información de empleados para asignar responsabilidades eficientemente.
System Parameterization
  • Como Administrador quiero configurar tipos de documentos para capturar correctamente la identidad de las personas.
  • Como Administrador quiero definir categorías de productos para organizar el catálogo de inventario.
  • Como Administrador quiero gestionar métodos de pago para ofrecer diferentes opciones de cobro en las ventas.
  • Como Administrador quiero definir tipos de movimientos de inventario para clasificar entradas y salidas.
Product & Inventory
  • Como Administrador quiero registrar productos con sus precios y stock inicial para controlar la oferta.
  • Como Empleado quiero registrar movimientos de inventario (entradas/salidas) para mantener el stock actualizado.
  • Como Usuario quiero consultar la disponibilidad de productos en tiempo real para informar a los clientes.
Sales Operations
  • Como Empleado quiero procesar ventas seleccionando productos y cantidades para registrar los ingresos.
  • Como Sistema quiero calcular automáticamente totales e impuestos para evitar errores humanos.
  • Como Empleado quiero generar facturas de venta para entregar un comprobante legal al cliente.
  • Como Empleado quiero registrar pagos asociados a las ventas para cerrar el ciclo comercial.
Customers & Services
  • Como Empleado quiero registrar clientes en el sistema para asociar sus compras y fidelizarlos.
  • Como Cliente quiero recibir mi factura por correo electrónico (opcional) para tener un respaldo digital.
Notifications
  • Como Administrador quiero crear plantillas de mensajes para notificar eventos de manera uniforme.
  • Como Sistema quiero enviar alertas de stock bajo para informar oportunamente la necesidad de reabastecimiento.
  • Como Sistema quiero enviar notificaciones de confirmación de pago para mantener informados a los clientes.

Priorización MoSCoW

Must

  • Como Usuario quiero iniciar sesión con usuario y contraseña para acceder de forma segura al sistema.
  • Como Administrador quiero definir roles básicos (administrador y vendedor) para controlar el acceso a las funcionalidades del sistema
  • Como Administrador quiero validar que los formularios críticos tengan datos obligatorios para evitar registros incompletos o erróneos.
  • Como Administrador quiero registrar productos con código, nombre, categoría, precio, stock y estado para administrar el inventario.
  • Como Administrador quiero editar y eliminar productos para mantener la información actualizada.
  • Como Administrador quiero consultar las existencias de productos en tiempo casi real para conocer el stock disponible y detectar alertas.
  • Como Administrador quiero registrar compras o ingresos de mercancía para incrementar el inventario
  • Como Sistema quiero actualizar automáticamente el inventario al registrar una venta para mantener el stock correcto.
  • Como Vendedor quiero registrar ventas asociando productos, cantidades y precios para generar el total de la venta.
  • Como Coordinador quiero asignar tripulación a un vuelo para garantizar que el vuelo esté atendido.
  • Como Vendedor quiero que cada venta quede registrada con fecha y usuario vendedor para mantener el control administrativo.
  • Como Administrador quiero generar reportes básicos de ventas por rango de fechas para analizar los ingresos del negocio.
  • Como Administrador quiero registrar clientes con datos básicos para asociarlos a las ventas cuando sea necesario.
  • Como Administrador quiero registrar proveedores con información básica para controlar el origen de los productos.

Should

  • Como Administrador quiero configurar permisos más específicos para el rol vendedor para controlar mejor el acceso a ciertas funcionalidades.
  • Como Administrador quiero restringir acciones sensibles como eliminación de registros para mantener la seguridad del sistema.
  • Como Administrador quiero generar reportes de productos con stock mínimo para planificar reposiciones oportunamente.
  • Como Administrador quiero analizar la rotación de productos para mejorar la gestión de compras.
  • Como Administrador quiero consultar el historial de ventas por cliente para analizar el comportamiento comercial.
  • Como Administrador quiero visualizar ventas por producto para identificar los artículos más vendidos y los de baja rotación.
  • Como Administrador quiero realizar ajustes manuales de inventario en caso de pérdida o daño para reflejar el stock real.
  • Como Administrador quiero consultar una bitácora de actividades para supervisar las acciones realizadas en el sistema.
  • Como Sistema quiero registrar automáticamente operaciones críticas para garantizar trazabilidad
  • Como Vendedor quiero aplicar filtros avanzados en el catálogo de productos para encontrar información rápidamente.
  • Como Administrador quiero exportar reportes en formato PDF o Excel para compartir información o conservar respaldo externo.

Could

  • Como Administrador quiero visualizar un panel de control con indicadores clave de ventas e inventario para tener una visión general del estado del negocio.
  • Como Administrador quiero ver alertas e información resumida en una sola vista para facilitar la toma de decisiones.
  • Como Administrador quiero recibir notificaciones visuales cuando un producto llegue a su nivel mínimo para actuar oportunamente
  • Como Vendedor quiero visualizar alertas de stock bajo al momento de registrar una venta para evitar vender productos agotados.
  • Como Administrador quiero organizar productos en categorías y subcategorías para mejorar la estructura del catálogo.
  • Como Vendedor quiero generar comprobantes de venta en formato ticket o factura para entregar un soporte formal de la transacción.
  • Como Administrador quiero autocompletado en búsquedas para mejorar la eficiencia operativa.
